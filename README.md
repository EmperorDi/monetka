# Монетка 🪙

Приватная локальная версия CoinKeeper — учёт личных финансов в стиле «перетащи монету».
PWA: устанавливается на iPhone через Safari, работает полностью офлайн, все данные
хранятся **только на устройстве** (IndexedDB) и никуда не отправляются.

## Возможности

- **Экран монет** — фирменный интерфейс CoinKeeper: перетащите монету
  *источник дохода → счёт* (доход), *счёт → категория* (расход), *счёт → счёт* (перевод).
  Тап по монете — быстрый ввод без перетаскивания.
- **Бюджеты** — лимит на месяц для категории, кольцо прогресса вокруг монеты
  (зелёное → оранжевое → красное при перерасходе).
- **Лента операций** по дням, редактирование и удаление по тапу.
- **Статистика** — доходы/расходы/разница, donut по категориям, динамика за 6 месяцев.
- **Повторяющиеся платежи** — еженедельные и ежемесячные, создаются автоматически
  при открытии приложения.
- **Экспорт/импорт** — резервная копия JSON, выгрузка операций в CSV (для Excel).
- Своя валюта, свои категории/счета/источники с эмодзи-иконками и цветами.

## Разработка

```bash
npm install
npm run dev        # http://localhost:5173
npm run build      # прод-сборка в dist/
npm run preview    # локальный просмотр сборки
```

Стек: Vite + React + TypeScript, Dexie (IndexedDB), vite-plugin-pwa (Workbox).

## Установка на iPhone

Service worker (офлайн-режим) требует HTTPS, поэтому сборку нужно выложить
на любой статический хостинг. Данные при этом остаются на телефоне — хостинг
раздаёт только код.

**Вариант А — Netlify Drop (без командной строки):**
1. `npm run build`
2. Откройте https://app.netlify.com/drop и перетащите туда папку `dist`.
3. Получите URL вида `https://….netlify.app`.

**Вариант Б — GitHub Pages (настроено):** приложение живёт в репозитории
https://github.com/EmperorDi/monetka и раздаётся с ветки `gh-pages`.
После изменений кода выполните `npm run deploy` — сборка и публикация одной командой.

**Затем на iPhone:**
1. Откройте URL в Safari.
2. «Поделиться» → «На экран “Домой”» → «Добавить».
3. Запустите «Монетку» с главного экрана — она работает офлайн и выглядит
   как нативное приложение.

> ⚠️ iOS может удалить данные сайтов, которыми долго не пользовались.
> Изредка делайте «Экспорт резервной копии» в настройках приложения.
